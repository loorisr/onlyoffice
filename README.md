# onlyoffice
Clean onlyoffice document server docker : https://github.com/ONLYOFFICE/Docker-DocumentServer/

Images available at: https://hub.docker.com/r/loorisr/onlyoffice

Based on **Ubuntu**

    Ubuntu 24.04
    PostgreSQL 16.11
    RabbitMQ 3.12.1

or **Debian**

    Debian Trixie 13
    PostgreSQL 17.6
    RabbitMQ 4.0.5

Size

- uncompressed is 2.82 vs 3.67 go (-23%)
